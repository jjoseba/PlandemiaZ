# PlandemiaZ
Minijuego en Unity sobre conspiranoicos y zombies, desarrollado en homenaje al 10º aniversario de la Marcha Zombie de Alcalá, que no pudo realizarse por la pandemia.

Puedes descargarlo desde itch.io:
